## Recent Work Completed

### Stats Calculation Fix (May 2, 2026)

Fixed the total listening time calculation in `statsStore.ts`:

1. **Position-based calculation** - Changed from wall-clock time to `(endPosition - startPosition) - skippedForward`
2. **Handle all Kx playback speeds** - Works for 1.5x, 1.75x, 2x, 3x etc.
3. **Seek handling** - seek_forward adds to skipped, seek_back updates position (no penalty)
4. **Complete events** - Now counts both pause and complete as session end

**Key files modified:**
- `web/src/stores/statsStore.ts` - aggregateDailyStats(), getEpisodeStats(), getTimeSaved()

### Episode Completion Fix

Fixed issue where completed episode would resume from previous position on re-login:
- `playerStore.ts` - Added `savePosition(currentEpisode.id, 0)` on episode completion to reset position in localStorage
