<task type="auto">
  <name>Task 5: Add theme toggle to header</name>
  <files>web/src/app/page.tsx</files>
  <action>
Add theme toggle button to the header in the home page.

Implementation:
1. Add toggleTheme function:
   ```typescript
   const [isDarkMode, setIsDarkMode] = useState(false);
   
   useEffect(() => {
     const savedTheme = localStorage.getItem('theme');
     const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
     setIsDarkMode(savedTheme === 'dark' || (!savedTheme && prefersDark));
   }, []);
   
   const toggleTheme = () => {
     const newIsDark = !isDarkMode;
     setIsDarkMode(newIsDark);
     if (newIsDark) {
       document.documentElement.classList.add('dark');
       localStorage.setItem('theme', 'dark');
     } else {
       document.documentElement.classList.remove('dark');
       localStorage.setItem('theme', 'light');
     }
   };
   ```
   
2. Add theme toggle button in header (after Logout button):
   ```tsx
   <button
     onClick={toggleTheme}
     className="p-2 text-gray-600 hover:text-gray-900 rounded-lg hover:bg-gray-100"
     title={isDarkMode ? 'Switch to light mode' : 'Switch to dark mode'}
   >
     {isDarkMode ? (
   ```
