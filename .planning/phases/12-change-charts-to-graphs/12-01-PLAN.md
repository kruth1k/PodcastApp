- Add responsive container
- Keep navigation controls (week/month/year)

### Task 3: Add Time Saved Feature

**Calculation Logic:**
- Track each listening session with its playback rate
- Time saved = (1 - 1/rate) × actual listening time
  - 1x speed = 0% saved
  - 1.5x speed = 33% saved
  - 2x speed = 50% saved

**Implementation:**
1. Update statsStore to capture playback rate with events
2. Calculate time saved in aggregate functions
3. Display on stats page

### Task 4: Update Stats Page

Add "Time Saved" card to the stats dashboard showing:
- Total time saved (in hours/minutes)
- Average playback speed used
