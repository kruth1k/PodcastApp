### Task 3: Add Time Saved Feature

**Calculation Logic:**
- Track wall clock time vs playback time per segment
- Each 10-second heartbeat interval: wall_time = 10s
- playback_time = 10 / playbackRate
- time_saved = wall_time - playback_time
- Example at 2x: 10s wall → 5s playback → 5s saved

**Implementation:**
1. Modify progress heartbeat to capture playback rate
2. Calculate time saved per 10s interval: saved = 10 * (1 - 1/rate)
3. Store accumulated time saved in statsStore
4. Display total time saved on stats page

### Task 4: Update Stats Page

Add "Time Saved" card to the stats dashboard showing:
- Total time saved (in hours/minutes)
- Average playback speed used
