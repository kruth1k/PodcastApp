### Task 3: Add Time Saved Feature

**Calculation Logic:**
- Only count time when audio is actually playing (not paused)
- For each playback session:
  - Track content time consumed (position changes)
  - Track wall clock time spent (play to pause duration)
  - time_saved = content_time - wall_clock_time
- Skip intervals where audio was paused (content position didn't change)

**Implementation:**
1. Add `getTimeSaved()` function to statsStore
2. Iterate through events:
   - Find play→pause or play→complete segments
   - Only include segments where content progressed
   - Calculate time_saved = (position_change) - (time_diff)
3. Sum all time_saved for total
4. Display on stats page

### Task 4: Update Stats Page

Add "Time Saved" card to the stats dashboard showing:
- Total time saved (in hours/minutes)
- Average playback speed used
