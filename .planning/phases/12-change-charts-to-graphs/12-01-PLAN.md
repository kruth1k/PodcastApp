### Task 3: Add Time Saved Feature

**Calculation Logic:**
- Compare content time vs wall clock time between progress events
- For each progress event pair: time_saved = (position_change) - (timestamp_diff/1000)
- Example: listen to 15s of content in 10s real time = 5s saved

**Implementation:**
1. Add `getTimeSaved()` function to statsStore
2. Calculate by iterating through progress events
3. time_saved = sum of (content_position_change - wall_clock_seconds) for each interval
4. Display total time saved on stats page

### Task 4: Update Stats Page

Add "Time Saved" card to the stats dashboard showing:
- Total time saved (in hours/minutes)
- Average playback speed used
