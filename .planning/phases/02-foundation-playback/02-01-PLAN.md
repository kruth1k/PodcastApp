  seek: (position: number) => void;

  skipBackward: () => void;

  <name>Task 1: Create PlayerStore with howler.js integration</name>
  <files>web/src/stores/playerStore.ts</files>
  <action>
Create Zustand store with howler.js integration for audio playback.

**playerStore.ts requirements:**
- Import Howl from 'howler' and create store with zustand
- State: currentEpisode (Episode | null), isPlaying (boolean), position (number), duration (number), playbackRate (number default 1), volume (number default 1), howlInstance (Howl | null)
- Actions:
  - playEpisode(episode): Creates new Howl with audio_url, stores episode, auto-plays
  - togglePlay(): Toggles play/pause on howl instance
  - seek(position): Calls howl.seek(position) - per D-02
  - skipForward(): Adds 15 seconds to current position - per D-03 (default 15s)
  - skipBackward(): Subtracts 30 seconds from current position - per D-04 (default 30s)
  - setPlaybackRate(rate): Sets howl.rate(rate) - supports 0.5x to 3x per D-05
  - updatePosition(): Called via interval, syncs howl position to state
- Use howl.on('end') to set isPlaying = false when episode finishes
- Use howl.on('load') to get duration
- Cleanup: In playEpisode, unload previous howl instance before creating new one
- Export store for use by PlayerBar

Reference: D-12 (howler.js), D-16 (separate playerStore), D-17 (howler.js events), D-18 (speed/skip intervals in store)
