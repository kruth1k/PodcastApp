## Problem 2: Episode Sorting

**Current behavior:**
- "Newest" shows newest episode of podcast (correct)
- "Oldest" shows oldest episode of current page only (incorrect)

**Expected:**
- "Oldest" should show oldest episode of podcast across all pages
- Page 1 = oldest, last page = newest

**Root cause:** Backend only sorts by `published_at.desc()` (newest first), no support for ascending order.

### Task 2: Add Sorting Support to Backend

**File:** `backend/app/crud.py`

**Change:** Add `sort_by` parameter to `get_episodes()`:
```python
def get_episodes(db: Session, podcast_id: str, offset: int = 0, limit: int = 50, sort_by: str = 'newest') -> List[models.Episode]:
    query = db.query(models.Episode).filter(models.Episode.podcast_id == podcast_id)
    if sort_by == 'oldest':
        query = query.order_by(models.Episode.published_at.asc())
    else:
        query = query.order_by(models.Episode.published_at.desc())
    return query.offset(offset).limit(limit).all()
```
