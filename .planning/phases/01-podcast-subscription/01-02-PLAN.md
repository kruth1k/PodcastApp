<task type="auto">
  <name>Task 1: Create RSS parsing service</name>
  <files>src/lib/rss-parser.ts</files>
  <action>
Create RSS parser using feedsmith library:

**parseFeed(url: string)** function:
- Fetch RSS feed from URL using native fetch
- Parse XML response with feedsmith
- Extract podcast metadata: title, description, imageUrl, author
- Extract episodes array: title, description, audioUrl, duration (seconds), publishedAt, guid
- Return { podcast: {...}, episodes: [...] }

**Handle iTunes extensions:**
- Extract itunes:image for podcast image
- Extract itunes:duration and convert to seconds (HH:MM:SS or seconds)
- Extract itunes:summary as description fallback

Return type:
```typescript
interface ParsedPodcast {
  title: string;
  description: string | null;
  feedUrl: string;
  imageUrl: string | null;
  author: string | null;
}

interface ParsedEpisode {
  title: string;
  description: string | null;
  audioUrl: string;
  duration: number | null;
  publishedAt: Date;
  guid: string;
}
```

Install feedsmith: npm install feedsmith
  </action>
  <verify>
<automated>npm install feedsmith && npx tsc --noEmit src/lib/rss-parser.ts</automated>
</verify>
  <done>RSS parser successfully fetches and parses real podcast feeds</done>
</task>
