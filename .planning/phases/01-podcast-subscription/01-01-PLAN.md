Create REST API endpoints:

**backend/app/crud.py:**
- create_podcast(podcast_data) -> Podcast
- get_podcasts() -> List[Podcast]
- get_podcast(podcast_id) -> Podcast | None
- delete_podcast(podcast_id) -> bool
- get_episodes(podcast_id) -> List[Episode]

**backend/app/routers/podcasts.py:**
- GET /api/podcasts - List all podcasts
- POST /api/podcasts - Add podcast (creates record, RSS parsing in plan 02)
- GET /api/podcasts/{id} - Get podcast details
- DELETE /api/podcasts/{id} - Unsubscribe

**backend/app/routers/episodes.py:**
- GET /api/episodes?podcast_id={id} - Get episodes for podcast

**Include routers in main.py**
