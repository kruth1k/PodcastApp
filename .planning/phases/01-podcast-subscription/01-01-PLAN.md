<task type="auto">
  <name>Task 1: Define Prisma database schema</name>
  <files>prisma/schema.prisma</files>
  <action>
Create Prisma schema with two models:

**Podcast model:**
- id: String (UUID)
- title: String
- description: String (nullable)
- feedUrl: String (unique)
- imageUrl: String (nullable)
- author: String (nullable)
- createdAt: DateTime
- updatedAt: DateTime
- episodes: Episode[] (relation)

**Episode model:**
- id: String (UUID)
- podcastId: String (foreign key to Podcast)
- title: String
- description: String (nullable)
- audioUrl: String
- duration: Int (seconds, nullable)
- publishedAt: DateTime
- guid: String (unique per podcast)
- createdAt: DateTime
- updatedAt: DateTime

Run `npx prisma generate` to create client. Run `npx prisma db push` to create SQLite database.
  </action>
  <verify>
<automated>npx prisma generate && npx prisma db push --accept-data-loss</automated>
</verify>
  <done>PrismaClient available, database tables created</done>
</task>

<task type="auto">
  <name>Task 2: Create podcast API endpoints</name>
  <files>src/app/api/podcasts/route.ts, src/app/api/podcasts/[id]/route.ts</files>
  <action>
Create two route files:

**GET /api/podcasts** - List all subscribed podcasts:
- Query prisma.podcast.findMany()
- Return array of podcast objects with episode count

**POST /api/podcasts** - Add new podcast:
- Accept {feedUrl} in body
- Validate feedUrl is valid URL format
- Create podcast record (RSS parsing happens in separate service later)
- Return created podcast with 201 status

**GET /api/podcasts/[id]** - Get single podcast:
- Find by UUID
- Include episodes ordered by publishedAt desc
- Return 404 if not found

**DELETE /api/podcasts/[id]** - Unsubscribe:
- Delete podcast (cascade deletes episodes)
- Return 204 on success, 404 if not found
  </action>
  <verify>
<automated>curl -X GET http://localhost:3000/api/podcasts returns 200</automated>
</verify>
  <done>All CRUD endpoints functional, returns proper status codes</done>
</task>

<task type="auto">
  <name>Task 3: Create episode API endpoint</name>
  <files>src/app/api/episodes/route.ts</files>
  <action>
Create GET /api/episodes endpoint:

**GET /api/episodes?podcastId={id}** - List episodes for a podcast:
- Require podcastId query parameter
- Query prisma.episode.findMany where podcastId equals parameter
- Order by publishedAt descending (newest first)
- Return array of episode objects
- Return 400 if podcastId missing, 404 if podcast not found
  </action>
